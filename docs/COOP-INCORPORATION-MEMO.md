# Cooperative Incorporation Decision Memo (Draft)

**Not legal advice.** Engage cooperative and securities counsel before filing or soliciting capital.

## Recommended direction

**Multi-stakeholder cooperative** (worker + consumer/member classes) in a state with a modern cooperative act (e.g. Colorado, Minnesota, Massachusetts, California).

## Decisions required

| Question | Options | Notes |
|----------|---------|-------|
| Statute | Worker co-op act vs general co-op act | Affects patronage and investor classes |
| Member classes | Worker / consumer / community | Voting weight per bylaws |
| Capital | Member shares vs external investors | See `MEMBER-CAPITAL-OFFERING-MEMO.md` |
| Tax | Subchapter T counsel review | |

## Filing package (counsel-prepared)

1. Articles of incorporation
2. Bylaws (patronage, voting, board, dissolution)
3. Initial member agreement
4. Data policy reference (`docs/DPIA.md`, `docs/PRIVACY-POLICY.md`)

## Timeline (indicative)

| Week | Milestone |
|------|-----------|
| 1–2 | Counsel engagement + statute selection |
| 3–6 | Draft articles/bylaws |
| 6–8 | File + EIN + bank |
